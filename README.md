# 🚍 Travel Behavior Analysis Project

## 🧠 Overview
This project analyzes travel behavior patterns of individuals based on their travel mode, purpose, budget level, and satisfaction. It helps understand how people choose transportation options in daily life and what factors influence their decisions.

The project is built using sample survey data and analyzed using Microsoft Excel.

---

## 🎯 Objectives
- Analyze preferred travel modes (Bus, Train, Car, Bike)
- Understand purpose of travel (Work, Education, Personal)
- Study budget distribution among travelers
- Measure satisfaction levels of users
- Identify patterns in travel behavior

---

## 📁 Dataset Information
- Total Records: 30 samples
- Features:
  - Age
  - Gender
  - Travel Mode
  - Purpose of Travel
  - Budget Level (Low, Medium, High)
  - Travel Frequency
  - Satisfaction Level

---

## 🛠 Tools Used
- Microsoft Excel
- Data Cleaning (Text to Columns)
- Data Analysis (COUNTIF, AVERAGEIF)
- Data Visualization (Pie Chart, Bar Chart, Column Chart)

---

## 📊 Analysis Performed
- Travel mode distribution analysis
- Purpose of travel analysis
- Budget-based travel behavior study
- Satisfaction level evaluation

---

## 📈 Key Insights
- Bus is the most commonly used travel mode
- Work-related travel is the highest category
- Most users prefer low-budget travel options
- Majority of users are satisfied with current travel options
- Car usage is higher among high-budget users

---

## 📊 Visualizations
The project includes the following charts:
